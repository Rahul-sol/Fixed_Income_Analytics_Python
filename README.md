# Fixed Income Analytics using Python

## Overview

This project performs fixed income analytics and bond valuation using Python.

It calculates:

- Bond Price
- Bond Price vs Yield Relationship
- Macaulay Duration
- Modified Duration
- Convexity
- Interest Rate Sensitivity Analysis

---

## Libraries Used

- NumPy
- Pandas
- Matplotlib

---

## Key Concepts

- Present Value
- Yield to Maturity
- Price-Yield Relationship
- Duration
- Modified Duration
- Convexity
- Interest Rate Risk

---

## Applications

- Fixed Income Analytics
- Treasury Management
- Asset Liability Management
- Interest Rate Risk Management
- Quantitative Finance
